    升级部署¶

    本版本，重构了配置文件，需要重新配置。

    1.安装脚本依赖。（若当前版本未升级至1.1.8，需执行！）

    pip3 install mysql-connector==2.2.9 SQLAlchemy==1.4.22

    2.重新拉取代码并修改配置文件。（若当前版本未升级至1.1.8，需执行！）

    cd /opt/opsany-paas/
git pull
cd install
cp install.config.example install.config
vim install.config

    请修改为之前部署时的相关配置。

    3.读取环境变量写入数据库。（若当前版本未升级至1.1.8，需执行！）

    cd /opt/opsany-paas/saas/
python3 add_env.py

    4.下载软件包到本地桌面，打开开发中心，进行手工上传部署

    根据邮件中的地址，下载opsany-saas.tar.gz，并解压到桌面，然后解压。

    本次部署完毕，导航栏会自动增加开发中心菜单，本次需要手工输入URL。请修改为您部署的域名或IP地址。

    https://192.168.56.11/saas/list/

    点击对应SAAS应用，点击部署，上传下载的对应的SAAS压缩包即可。
